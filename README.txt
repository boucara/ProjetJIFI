Le but de ce projet est de pouvoir s'entrainer sur l'utilisation de differentes technologies et langages web. 
L'idée est la réalisation d'une application web à partir de la quelle on pourrait: 
-Consulter les ménus d'un Restaurant 
-Passer des commandes , à la suite de la quelle on sera livrer chez sois.
Les technologies utilisées:
- IDE :Eclipse
J2EE
-Serveur: JETTY
-Angular JS coté client
-EJB
-JPA
-Base de donnée : ORACLE
